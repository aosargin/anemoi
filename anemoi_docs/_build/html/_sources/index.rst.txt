Welcome to anemoi's documentation!
==================================

Anemoi were wind gods who were each ascribed a cardinal direction from which their respective winds came, and were each associated with various seasons and weather conditions. This Python package is intended to analize wind data for the renewable energy industry. 

**Requirements:** 

* Python 3
* matplotlib
* numpy
* pandas
* pyarrow
* pyodbc
* requests
* scipy
* seaborn
* statsmodels
